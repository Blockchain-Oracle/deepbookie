# DeepBookie — Milestones

What's shipped, what's building, what's on the roadmap. Live version: [docs.deepbookie.xyz/milestones](https://docs.deepbookie.xyz/milestones).

## Now — shipped and live on Sui testnet

- **Live web app** — chat to trade DeepBook at [deepbookie.xyz](https://deepbookie.xyz).
- **44 tools** — the full DeepBook surface: Predict (yes/no BTC markets) + Spot (the order book).
- **5 npm packages** — `@deepbookie/predict-client`, `core`, `node`, `mcp`, `cli`.
- **Claude skill** — install in one line: `npx skills add Blockchain-Oracle/deepbookie --skill deepbookie`.
- **MCP server** — trade from Claude Desktop, Cursor, or any MCP client.
- **Terminal CLI** — `npx @deepbookie/cli`.
- **Docs site** — [docs.deepbookie.xyz](https://docs.deepbookie.xyz).
- **First Predict TS client** — no one had shipped one for DeepBook Predict.

## Next — building

- **Mainnet** — move from testnet to real trades on Sui mainnet.
- **Mobile app** — iOS + Android.
- **Telegram & Discord bot** — trade without leaving your group chat.

## Later — on the roadmap

- **Voice trading** — talk to DeepBookie, don't type.
- **More markets** — ETH, SOL, and beyond.
- **Agent strategies** — auto-trade within limits you set (stop-loss, take-profit, scheduled bets).
- **Embeddable SDK** — drop DeepBookie's chat into any Sui dapp in a few lines.

---

The rule for what we build next: *does it help more people trade DeepBook?* If something you want isn't here, [open an issue](https://github.com/Blockchain-Oracle/deepbookie/issues).
